This repository contains the code for the [TeamPlayer.com](https://teamplayer.com) domain-for-sale landing page.

## Purpose

This site serves as a landing page to announce that TeamPlayer.com is available for purchase and to collect inquiries from interested buyers.

## Features

- Clean, modern, and mobile-friendly single-page design
- Clearly advertises that TeamPlayer.com is for sale
- Integrated inquiry form (compatible with Formspree or your preferred provider) for collecting offers and questions
- Automatically detects and displays the current domain name on the page and in form submissions
- Fast, secure, and easy to deploy (static HTML, CSS, and JavaScript)
- Easily customizable for your branding and content needs

## Usage

1. **Clone or download this repository.**
2. **Edit the content** in `index.html` as needed (domain name, form endpoint, features, etc.).
3. **Deploy to your preferred static web host** (Netlify, Vercel, GitHub Pages, cPanel, etc.).
4. **Update DNS settings** to point TeamPlayer.com to your hosting provider.

## Customization

- Update the domain name in the header, title, and throughout the page as necessary
- Set the Formspree endpoint or other contact integration to receive form submissions
- Adjust design, features, or text to suit your preferences or branding

## DNS Setup

For best results, point TeamPlayer.com to your hosting provider using a CNAME, ALIAS/ANAME, or A record as appropriate.  
Avoid HTTP redirects, as these can interfere with domain detection and inquiry tracking.
